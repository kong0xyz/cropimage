# Google Ads

## Get Started

1. Go to [Google Ads](https://ads.google.com/home/)
2. Click on "Start"
3. Click on "Create a new campaign"
4. Click on "Search"
5. Click on "Create a new campaign"

## Update ads.txt

Open `ads.txt` file and update the content.

```
google.com, pub-1234567890, DIRECT, f08c47fec0942fa0
```
