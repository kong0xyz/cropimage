# Google Adsense

To start earning money with AdSense, you need to add payment information and link your website.

## Get Started

1. Go to [Google Adsense](https://adsense.google.com/)
2. Click on "Start using"
3. Login...

## Add a website

1. Click on "Add a website", input domain, click "Continue"

## Verify website ownership

1. Select Ads.txt code segment (ads.txt is a separate file, easy to maintain; you can also choose other ways)

2. Update `public/ads.txt` (the same Google Adsense account is shared, usually unchanged)

```txt
google.com, pub-[your-adsense-id], DIRECT, f08c47fec0942fa0
```

3. Click on "Verify ownership"

### Apply for review

1. Click on "Apply for review"

2. Click on "Submit"

3. Wait for review

(Done)

# Google Adsense 谷歌广告

借助 AdSense 开启创收之路，需要添加收款信息并关联网站。

## 开始接入

### 开始

1. 访问 [Google Adsense](https://adsense.google.com/)
2. 点击「开始使用」
3. 登录...

### 添加网站

1. 点击「添加一个网站」，输入域名，点击「继续」

### 验证网站所有权

1. 选择 Ads.txt 代码段（ads.txt这种是独立文件，易维护；也可以选其他方式）

2. 更新网站根目录，源码 `public/ads.txt`（同一个 Google Adsense 账号是共用的，一般不变）

```txt
google.com, pub-[your-adsense-id], DIRECT, f08c47fec0942fa0
```

3. 勾选「我已发布 ads.txt 文件」点击「验证」
4. 您的网站已通过验证，点击「下一步」

### 申请审核

1. 点击「申请审核」

2. 等待审核
