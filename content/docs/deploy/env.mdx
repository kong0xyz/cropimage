## 基本配置

- `NEXT_PUBLIC_SITE_URL` - The root URL of the site. e.g. `https://[your-domain]`

## Badges

- `NEXT_PUBLIC_FAZIER_LAUNCH_SLUG` - Fazier Launch Slug
- `NEXT_PUBLIC_STARTUPFA_LAUNCH_SLUG` - StartupFa Launch Slug
- `NEXT_PUBLIC_PRODUCTHUNT_LAUNCH_SLUG` - Product Hunt Launch Slug
- `NEXT_PUBLIC_TWELVE_TOOLS_LAUNCH_SLUG` - Twelve Tools Launch Slug (true or null)
- `NEXT_PUBLIC_TURBO0_LAUNCH_SLUG` - Turbo0 Launch Slug
- `NEXT_PUBLIC_MAGICBOX_TOOLS_LAUNCH_SLUG` - MagicBox (true or null)

## Analytics

- `GOOGLE_ANALYTICS_ID` - Google Analytics ID (Measurment ID)
- `NEXT_PUBLIC_ADSENSE_URL` - Google AdSense URL

## Database 

- `DATABASE_URL` Drizzle ORM DB URL

## Better Auth

- `NEXT_PUBLIC_BETTER_AUTH_URL`  比如: `https://saas.kong0.xyz`
- `BETTER_AUTH_SECRET`

第三方 OAuth 2.0 登录配置详见: [Better Auth](/docs/deploy/better-auth)

- `BETTER_AUTH_GITHUB_CLIENT_ID` Github Client
- `BETTER_AUTH_GITHUB_CLIENT_SECRET`

- `BETTER_AUTH_GOOGLE_CLIENT_ID` Google Client
- `BETTER_AUTH_GOOGLE_CLIENT_SECRET`

- `BETTER_AUTH_DISCORD_CLIENT_ID` Discord Client
- `BETTER_AUTH_DISCORD_CLIENT_SECRET`

- `BETTER_AUTH_MICROSOFT_CLIENT_ID` Microsoft Client
- `BETTER_AUTH_MICROSOFT_CLIENT_SECRET`

## Resend
- `RESEND_API_KEY`
- `RESEND_FROM_EMAIL` 发件人邮箱名称地址。比如: `Next SaaS <support@saas.kong0.xyz>`

## Stripe

配置和获取详见：[Payment with Stripe](/docs/int/stripe)

- `STRIPE_SECRET_KEY` 支付密钥
- `STRIPE_WEBHOOK_SECRET` 回调密钥

- `STRIPE_BASIC_PRICE_ID` 基础月费价格ID
- `STRIPE_BASIC_ANNUAL_PRICE_ID` 基础年费价格ID
- `STRIPE_PRO_PRICE_ID` 专业月费价格ID
- `STRIPE_PRO_ANNUAL_PRICE_ID` 专业年费价格ID
