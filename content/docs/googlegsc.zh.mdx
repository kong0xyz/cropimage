# Google Search Console

## 添加网站

1. 在 Google Search Console 中，点击 "添加网站"
2. 输入网站的 URL
3. 点击 "添加"
4. 完成所有权验证

## 站点地图

1. 添加新的站点地图 `https://www.[your-domain]/sitemap.xml`
