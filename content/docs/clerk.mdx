## Introduction

Clerk is a powerful authentication solution that allows you to add authentication to your application in minutes.

## Installation

```bash
npm install @clerk/nextjs
```

## Configuration

```bash
    clerk init
```

## Usage

```bash
    import { SignInButton } from "@clerk/nextjs";

    <SignInButton />
```

## Customization

```bash
    clerk customize
```
