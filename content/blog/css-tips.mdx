# CSS 高级技巧与最佳实践

CSS是前端开发的核心技术之一，掌握一些高级技巧可以让你的开发更加高效。

## 1. CSS Grid 布局技巧

### 自适应网格布局
```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  gap: 1rem;
}
```

### 网格区域命名
```css
.layout {
  display: grid;
  grid-template-areas: 
    "header header header"
    "sidebar content content"
    "footer footer footer";
  grid-template-rows: auto 1fr auto;
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.content { grid-area: content; }
.footer { grid-area: footer; }
```

## 2. Flexbox 高级用法

### 完美居中
```css
.center {
  display: flex;
  align-items: center;
  justify-content: center;
}
```

### 自动外边距技巧
```css
.nav {
  display: flex;
}

.nav-item:last-child {
  margin-left: auto;
}
```

## 3. CSS 自定义属性

### 主题切换
```css
:root {
  --primary-color: #3b82f6;
  --text-color: #1f2937;
  --bg-color: #ffffff;
}

[data-theme="dark"] {
  --primary-color: #60a5fa;
  --text-color: #f9fafb;
  --bg-color: #111827;
}

.button {
  background: var(--primary-color);
  color: var(--bg-color);
}
```

## 4. 现代CSS选择器

### :has() 选择器
```css
/* 选择包含图片的卡片 */
.card:has(img) {
  border-color: var(--primary-color);
}

/* 选择没有子元素的容器 */
.container:not(:has(*)) {
  display: none;
}
```

### 容器查询
```css
@container (min-width: 400px) {
  .card {
    display: grid;
    grid-template-columns: 1fr 2fr;
  }
}
```

## 5. 性能优化技巧

### 硬件加速
```css
.animated {
  transform: translateZ(0);
  will-change: transform;
}
```

### 关键渲染路径优化
```css
/* 内联关键CSS */
.above-fold {
  display: block;
  margin: 0 auto;
  max-width: 1200px;
}
```

## 总结

掌握这些CSS技巧可以：
- 🚀 提高开发效率
- 💡 创造更好的用户体验
- 🔧 减少代码冗余
- 📱 增强响应式设计
