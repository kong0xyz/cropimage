# 欢迎来到我的博客

欢迎访问我的全新博客！这是使用 Next.js 15 和 MDX 构建的现代化博客系统。

## 功能特性

这个博客系统包含以下特性：

- **🚀 高性能**：基于 Next.js 15 的静态生成
- **📱 响应式设计**：完美适配各种设备
- **🏷️ 标签系统**：轻松组织和查找内容
- **📂 分类管理**：按主题分类文章
- **🔗 社交分享**：一键分享到各大社交平台
- **🌐 国际化支持**：多语言支持

## 写作体验

使用 MDX 格式让我能够：

1. 编写 Markdown 内容
2. 嵌入 React 组件
3. 添加交互元素
4. 保持代码高亮

```javascript
// 示例代码
function BlogPost({ title, content }) {
  return (
    <article>
      <h1>{title}</h1>
      <div>{content}</div>
    </article>
  );
}
```
